Core

PersistenceManager

sair.aiagent.core · 612 行

SQLite 统一持久化核心。单文件数据库 aiagent.db，WAL 模式，FTS5 全文搜索，替代 5 个独立 JSON 文件。5 张表：memories / memories_fts / journal / conversations / app_state。事务安全，旧数据自动迁移。

Core

AiConfig

sair.aiagent.core · 372 行

单例配置管理器。API Key AES-128 加密存储，config.properties 持久化。提示词通过 PromptManager 统一管理，load 时追加到 PromptManager，save 时从 PromptManager 读取 extra 写入。

Core

PromptManager

sair.aiagent.core · 358 行

集中式提示词管理器（单例）。三段提示词统一入口：systemPrompt（控制台Chat）、execqPrompt（QQ通道）、agentStaticPrompt（Agent执行）。硬编码默认值 + config.properties 追加模式持久化。setSystemPrompt/setExecqPrompt 走追加写入，重启不丢失。

Core

AgentExecutor

sair.aiagent.core · 1184 行

ReAct 推理-行动循环引擎。最多 30 轮循环，execq 通道 50 轮。15 种 XML 标签：cmd/readfile/readdir/sys/evaljs/eval/web/remember/download/superise/editprompt/stop/sendimage/sendrecord/sendfile。execq 受限白名单（cmd/web/readdir/setname/stop），自动绕过确认门控。buildStaticPrompt() 通过 PromptManager 获取模板并动态填充 {os}/{shell}/{jdk}/{plugins}。

Core

ActivityActions

Core

DeepSeekClient

Core

DynamicCodeEngine

sair.aiagent.core · 353 行

双引擎动态代码执行：Nashorn JavaScript 即时执行 + JavaCompiler 即时编译注入。InMemoryClassLoader 加载字节码，编译→加载→执行→丢弃，全程一次性。支持动态 .jar 加载。

Core

EmotionManager

sair.aiagent.core · 268 行

AI 情绪状态机。情绪词典：辱骂词 17 个(-20)、夸赞词 18 个(+15)、安慰词 18 个(+25)。阈值：happiness≤-30 或连败≥3→暂停；≥95 且连夸≥2→彩蛋(v2.0 阈值 80→95)。持久化到 SQLite app_state。

Core

StreamPrinter

sair.aiagent.core · 280 行

单例流式输出核心。LinkedBlockingQueue FIFO 队列，双通道：行输出(无延迟) + 字符输出(10ms 打字机效果)。start() 先排空旧队列再启动新流。守护线程永续运行。

Core

SysConsoleExecutor

sair.aiagent.core · 248 行

跨平台系统命令执行器。ProcessBuilder 封装，自动识别 cmd/sh 切换，30s 超时。支持同步执行和实时回调（OutputListener）。Windows 强制 UTF-8 编码。

Core

MemoryManager

sair.aiagent.core · 85 行

持久化记忆管理器 SQLite 门面。委托 PersistenceManager 的 FTS5 全文搜索 + CRUD。支持添加/列出/删除/清空，FTS5 按 BM25 相关性排序，失败回退 LIKE。上下文上限 2000 字符。

Core

JournalManager

sair.aiagent.core · 47 行

Core

ConversationHistory

sair.aiagent.core · 148 行

对话历史管理器，滑动窗口 + SQLite 持久化。支持约 900K token 上下文窗口。Token 估算：中文~0.6 token/字，英文~0.25 token/字。超限自动移除最旧消息，至少保留 4 条。

QQMessageHandler

sair.aiagent.onebot · 1373 行

QQ 消息处理核心 —— 消息解析、过滤、路由到 AI Agent，好友/群邀请好感度决策。无依赖 JSON 解析(JsonUtil)。消息过滤：群消息仅响应 @机器人 或提到名字。execq 通道：独立提示词、统一记忆库。群管标签处理：processQqActions() 在发送前拦截 AI 响应中的 10 种群管标签(bankick/muteall/setadmin/setcard/setgroupname/leavegroup/block/unblock/delfriend)，OneBot 层自有能力，不污染 AgentExecutor 主线。
