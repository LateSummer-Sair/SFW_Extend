项目概述

AiAgent 是一个为 SFW (SairFrameWork) 框架打造的 AI 智能代理插件（v1.8），全面对接 DeepSeek API，实现流式对话与自主任务执行。新增 QQ OneBot v11 机器人集成，execq 独立通道（插件白名单门控），**统一记忆库架构**（所有对话历史统一管理，通过来源标记区分群聊/私聊，实现记忆联通）。26 个 Java 源文件，约 7,200 行代码，SQLite 全局持久化，外部依赖：SFW.jar + gson-2.8.6.jar + sqlite-jdbc-3.42.0.0.jar。

  • ✓
    QQ OneBot v11 机器人（统一记忆）
    反向 WebSocket 连接接手 QQ 私聊群聊。execq 独立通道（独立提示词/记忆/上下文），仅开放 cmd/web/readdir/setname 标签，<cmd> 插件白名单门控，自动绕过确认。**统一记忆库架构**：所有对话存储在 unified_memory.db，通过 source_type/source_id 标记区分群聊/私聊，实现记忆联通，AI 不会失忆。@回复/引用/提到名字触发响应，支持主动查看群聊（每5分钟）
  • ✓
    AI 接管 SFW
    遍历 Libraries.activities 收集所有插件 help，AI 自主分析并执行命令
  • ✓
    流式逐字输出
    单例 FIFO 队列逐字输出 + 10ms 延迟，打字机效果 + 多颜色区分，新流自动排空旧队列
  • ✓
    ~900K Token 上下文
    对话历史滑动窗口，自动估算 token 并裁剪，保留至少 4 条消息。持久化到 aiagent.db (SQLite)
  • ✓
    情绪陪伴系统
    AI 具备情绪状态机：被夸开心、被骂难过、连续失败撒娇暂停、开心到爆触发彩蛋弹窗
  • ✓
    SQLite 统一持久化
    单文件 aiagent.db 替代 5 个 JSON，WAL 模式 + FTS5 全文搜索 + 自动迁移旧数据，支持记忆/日志/对话/情绪/上下文全量持久化
  • ✓
    动态代码注入与执行
    Nashorn JavaScript 即时执行 + JavaCompiler 即时编译注入，编译→加载→执行→丢弃，全程一次性
  • ✓
    联网搜索 + 文件下载
    HTTP GET 获取网页内容（SSRF 内网黑名单防护），支持搜索引擎/API调用/文件下载
  • ✓
    跨平台系统终端
    ProcessBuilder 封装，Windows/Linux/macOS 自适应 cmd/sh 切换，UTF-8 输出，30s 超时
  • ✓
    三模式记忆同步
    chat ↔ exec ↔ execs 双向上下文注入：Agent 结果注入 Chat + Chat 历史注入 Agent + Agent 层层递进
  • ✓
    FTS5 全文记忆搜索
    SQLite FTS5 全文搜索引擎 + LIKE 回退，按相关性排序检索记忆，自动注入 Chat/Agent 上下文
  • ✓
    高危操作确认门控
    ConfirmationGate 统一门控：动态注入、系统命令、文件读取等 8 类操作需 ai/yes ai/no 确认，60s 超时自动拒绝
  • ✓
    AI 自我进化
    AI 可通过 <editprompt> 标签修改自身 system prompt，逐步演化性格
  • ✓
    增量持久化写入
    INSERT/UPDATE 逐条写入替代全量 Gson.toJson() 覆盖，事务保护 + busy_timeout=3000ms，高并发安全
  • ✓
    配置持久化 + AES 加密
    API Key 使用 AES 加密存储到 config.properties，URL/Model/Prompt 自动持久化，旧值自动升级
