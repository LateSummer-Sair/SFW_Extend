项目概述

AiAgent 是一个为 SFW (SairFrameWork) 框架打造的 AI 智能代理插件（v2.0），全面对接 DeepSeek API，实现流式对话与自主任务执行。集成 QQ OneBot v11 机器人，支持群聊/私聊双通道，execq 独立通道（插件白名单门控），统一记忆库架构（所有对话历史统一管理，通过来源标记区分群聊/私聊，实现记忆联通）。具备好友/群邀请好感度决策、反滥用保护、群上下文感知、<stop> 紧急停止标签、群管理违规处罚（警告/禁言/踢出/拉黑）等高级功能。33 个 Java 源文件，约 11,147 行代码，SQLite 全局持久化，外部依赖：SFW.jar + sqlite-jdbc-3.42.0.0.jar + gson-2.8.6.jar。

  • ✓
    QQ OneBot v11 机器人（统一记忆 + 群管理）
    反向 WebSocket 连接接手 QQ 私聊群聊。execq 独立通道（独立提示词/记忆/上下文），仅开放 cmd/web/readdir/setname/stop 标签，<cmd> 插件白名单门控。**统一记忆库架构**：所有对话存储在 unified_memory.db，通过 source_type/source_id 标记区分群聊/私聊。群聊中 AI 拥有完整群管理能力（警告/禁言/踢出/拉黑），通过 NapCatApi 调用 NapCat QQ 框架标准 API 执行违规处罚。@回复/引用/提到名字触发响应，支持主动查看群聊（每5分钟）
  • ✓
    AI 接管 SFW

  • ✓

    单例 FIFO 队列逐字输出 + 10ms 延迟，打字机效果 + 多颜色区分，新流自动排空旧队列
  • ✓

    对话历史滑动窗口，自动估算 token 并裁剪，保留至少 4 条消息。持久化到 aiagent.db (SQLite)
  • ✓

    7步优先级决策链智能处理群邀请：①主人无条件接受 ②好友申请通过后5分钟内拉群→拒绝+警告 ③好感度>600→接受 ④好感度≤200→拒绝 ⑤好感度200-600→忽略。好友申请验证：无消息→拒绝，有消息→接受。反滥用5分钟窗口防止恶意拉群
  • ✓

    单文件 aiagent.db 替代 5 个 JSON，WAL 模式 + FTS5 全文搜索 + 自动迁移旧数据，支持记忆/日志/对话/情绪/上下文全量持久化
  • ✓

    Nashorn JavaScript 即时执行 + JavaCompiler 即时编译注入，编译→加载→执行→丢弃，全程一次性

    FTS5 全文记忆搜索

  • ✓

    AI 具备情绪状态机：被夸开心、被骂难过、连续失败撒娇暂停、开心到爆触发彩蛋弹窗。v2.0 优化彩蛋频率（阈值80→95，触发后消耗情绪值），防止过度频繁
  • ✓

    AI 可在QQ消息中发送 <stop> 标签让 Agent 立即停止执行。同时支持 QQ 消息直接检测（主人发送含 <stop> 的消息），以及 execq Agent 内部输出 <stop> 标签自停。三通道全覆盖：QQ直接检测 + execq AI输出 + 控制台TAG_PATTERN
  • ✓
    高危操作确认门控
    ConfirmationGate 统一门控：动态注入、系统命令、文件读取等 8 类操作需 ai/yes ai/no 确认，60s 超时自动拒绝
  • ✓

  • ✓
    增量持久化写入
    INSERT/UPDATE 逐条写入替代全量 Gson.toJson() 覆盖，事务保护 + busy_timeout=3000ms，高并发安全

    API Key 使用 AES 加密存储到 config.properties，URL/Model/Prompt 自动持久化，旧值自动升级
